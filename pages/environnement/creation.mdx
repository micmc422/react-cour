# Création d'un Nouveau Projet React

Une fois Node.js et npm installés, nous allons créer un nouveau projet React. Nous utiliserons Create React App, un outil officiellement supporté par l'équipe de React, qui configure votre projet avec des réglages par défaut optimisés pour la plupart des applications.

Voici comment procéder :

1. Ouvrez une nouvelle fenêtre de terminal.

2. Naviguez vers le répertoire où vous souhaitez créer votre nouveau projet React. Par exemple, si vous voulez créer votre projet dans un répertoire appelé "mon-projet-react" sur votre bureau, vous pouvez utiliser la commande suivante :

```bash
cd ~/Desktop
```

3. Une fois que vous êtes dans le répertoire approprié, vous pouvez créer un nouveau projet React en utilisant la commande suivante :

```bash
npx create-react-app mon-projet-react
```

Remplacez "mon-projet-react" par le nom que vous souhaitez donner à votre projet.

4. Cette commande va créer un nouveau répertoire avec le nom de votre projet, et y installer toutes les dépendances nécessaires pour un projet React. Cela peut prendre quelques minutes.

5. Une fois l'installation terminée, vous pouvez démarrer votre application React en utilisant la commande suivante :

```bash
cd mon-projet-react
npm start
```

Cela va démarrer un serveur de développement local et ouvrir votre application React dans votre navigateur par défaut.

Maintenant, vous êtes prêt à commencer à développer votre application React !
