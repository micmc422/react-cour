# Installation et Configuration de l'Environnement de Développement

## Étape 1: Installation de Node.js et npm

Node.js est un environnement d'exécution JavaScript qui permet d'exécuter du code JavaScript côté serveur. npm, qui est le gestionnaire de paquets de Node.js, est essentiel pour gérer les dépendances de votre projet React.

Suivez ces étapes pour installer Node.js et npm:

1. Visitez le site officiel de Node.js à l'adresse suivante : https://nodejs.org/
2. Vous verrez deux versions disponibles pour le téléchargement : LTS (Long Term Support) et Current. La version LTS est recommandée pour la plupart des utilisateurs car elle est plus stable. Cliquez sur le bouton vert pour télécharger cette version.
3. Une fois le fichier d'installation téléchargé, exécutez-le. Suivez les instructions à l'écran pour installer Node.js et npm.
4. Une fois l'installation terminée, vous pouvez vérifier si Node.js et npm sont correctement installés en ouvrir une nouvelle fenêtre de terminal et en tapant les commandes suivantes :

```bash
node -v
npm -v
```

Ces commandes afficheront les versions de Node.js et npm respectivement. Si ces commandes renvoient des numéros de version, cela signifie que Node.js et npm sont correctement installés.

Notez que l'installation de Node.js installe également npm, donc vous n'avez pas besoin de l'installer séparément.
